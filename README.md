Цель: проверить, способен ли разработчик быстро организовать слой данных, организовать симметрию между API и admin-интерфейсом, обработать консистентность, и при этом выбрать правильный уровень абстракции для управления тестовыми данными — это критичный навык для построения тестируемых и масштабируемых систем.
